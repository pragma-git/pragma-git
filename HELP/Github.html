What is Github

Github is a remote git-repository. You may connect your local repository to this, and can use this as a backup, to keep different computers you work with in sync, or for cooperation with others.

Setup

Github requires an account (that is, a username), and a password. The location of the remote github repository is defined by an URL, of a format similar to "https://...github.com..". This window helps you getting this correct.

A new account can be defined from the github website, by following the link "Register a Github account".

An access token can also be defined from the github website, by following the link "Create an access token". The access token can act as a password for git, and give access to operations such as creating a new repository from within Pragma-git.

When creating an access token fill in the form on the github web-page (opened from the "Create an access token" link) :

The URL

This form in this window helps you to compose the correct URL from fields for username, access token, and existing repository. A generated URL preview is shown below these fields.

Create a Remote Github Repository

As said before, this can be done via github's website. However, this window makes the process simpler. The fields in this section are:

Private repository
a checkbox that makes the repository visible to only you and others who you have given your username, password and access token.
Name of New Repository
is obvious. This field inhibits forbidden characters to be typed, and exchanges others to allowed characters. This must be a new repository name, that does not already exist on your github account.
Description
is a short text describing your repository, displayed in the github web-view.

button creates a new repository in you account (requires correct username and access token fields).

Finish the Remote Repository creation

A status message is shown below the buttons, displaying success or failure. When succesful, the repository name is copied to the "Existing Repository Name" field.

button closes this window and updates the URL in the settings window.

button (higher up on the page) leaves the window keeping the old URL in the settings window.
