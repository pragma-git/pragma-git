Current branch

The commit tree shows commits on the current branch in green , and commits on other branches without color . Note, the correct branch can only be deduced if the recommended settings "Merge without fast-forward" is used.

Navigating

Moving the mouse pointer over the commit message, a grey shaddow appears, together with a checkbox .

Select commit

Clicking on the commit makes the text green, and moves history to the selected commit. The main window is updated to display this commit. The main window can be used as before, looking for changes, clicking on the modified files to get a list of changes to prior commit, etc.

Compare with fixed commit

Clicking the checkbox , makes the checked commit a fixed historical commit ().

Selecting the text of any other commit (not the checkbox) moves history to the newly selected commit (both in graph and in main window). The bottom of the main window is updated to display number of modified files between these two commits (exactly as using the checkbox and the arrow navigation directly in the main window).
